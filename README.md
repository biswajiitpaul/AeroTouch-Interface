AeroTouch Interface

Status
Tech
UI

AeroTouch Interface is a touch-free system that allowed me to test out the use of hand gestures rather than a mouse or keyboard.

It takes advantage of MediaPipe, which tracks hands in real-time and translates simple gestures into on-screen interactions. The project will also consist of a modern dark interface with fluid effects, particle and simple sound feedback to give it a more futuristic look.

Features

* Real-time hand tracking
* Gesture-based interaction
* Modern dark UI
* Clean particles and animations.
* Interactive visual feedback
* Does not require installation.

Sandbox and Language Stack

* HTML
* CSS
* JavaScript
* MediaPipe Hands
* Canvas API
* Web Audio API

How to Run

git clone https://github.com/biswajiitpaul/aerotouch-interface.git
cd aerotouch-interface

CLASSIC Enter open index.html into your browsers and permit camera access.

About

I constructed this project to learn about touchless interaction and to see how systems based on gestures operate. The aim was to build something interactive and futuristic with with their bare web technologies.
